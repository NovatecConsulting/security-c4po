This tool allows a security tester to keep track of the testing progress after OWASP ASVS 4.0. At this point, the ASVS testing materials are mostly available as Excel and pdf files which have to be sent back and forth with the client -- what a tedious task. Using a "webified" version of the testing guide would be more fun, right? In the current version, only an outline is implemented to give an idea of what this tool is supposed to become.


Planned features:
  • Have fun developing a full-stack web app with modern technologies!
  • Allow sharing the current progress with the customer.
  • Have different user roles and rights to view and modify data.
  • Upload screenshots of findings.
  • Have automated scanners installed that help the tester get started.
  • ...
